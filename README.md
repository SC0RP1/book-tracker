# Трекер прочитанных книг

Консольное приложение для учета прочитанных книг с модельной архитектурой.

## Описание

Приложение позволяет вести учет прочтинных книг, сохраняя информацию в JSON-файле. Поддерживает добавление, удаление, 
просмотр книг, а также расчет статистики.

## Структура модулей

- **main.py** — точка входа и пользовательский интерфейс (меню, ввод-вывод)
- **storage.py** — чтение, запись и управление данными в JSON (CRUD операции)
- **stats.py** — расчет средней оценки и статистики по авторам
- **models.py** — класс Book (автор, название, оценка, дата)
